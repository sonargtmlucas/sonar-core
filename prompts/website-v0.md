# Sonar GTM Website — v0/Lovable Prompt

Paste this entire prompt into v0.dev or Lovable to generate sonargtm.com.

---

Build a dark, premium marketing website for Sonar GTM — a signal-based GTM intelligence company. This is NOT a dashboard. It's a landing page that converts VP Sales / CRO / Founders into booked calls.

## Brand and visual system

Colors (use exactly these hex values):
- Background: #071A2B (Abyss Navy — very dark blue)
- Primary accent: #1CB7FF (Signal Blue — the one color that pops)
- Body text: #F7FAFC (Polar White)
- Muted text: #B8C4CE (Sonar Silver)
- Borders / dividers: #334452 (Graphite)
- Alert / CTA pulse: #FF6B57 (Pulse Coral — use sparingly)

Typography:
- Headlines: Sora (Google Font) — Bold / ExtraBold
- Body: Inter (Google Font) — Regular / Medium
- Monospace accents (for signal data, stats): DM Mono

Visual style:
- Dark, cinematic, premium — NOT hacker / crypto / military
- Use concentric rings / pulse animations to represent signal propagation (the Sonar radar concept)
- Negative space — breathing room between sections
- One active signal point in the logo (a cyan dot with pulse rings around it)
- No crosshairs, no weapons imagery, no robot faces

## Logo
SVG concentric ring logo:
- 3 rings of increasing radius, opacity decreasing outward (0.8 → 0.4 → 0.15)
- Center dot in Signal Blue (#1CB7FF)
- One sweep line from center to upper-right (like a radar sweep)
- Word "SONAR" in Sora ExtraBold next to it

## Technical requirements
- Next.js 14 with TypeScript
- Tailwind CSS
- Framer Motion for animations (pulse rings, fade-in on scroll, page transitions)
- Fully responsive (mobile-first)
- No backend needed — static site, CTA links to Calendly
- SEO: title "Sonar GTM — Signal-Based Revenue Intelligence", description "Sonar helps B2B sales teams identify buying windows and act before the competition."

## What NOT to include
- No chatbot or AI widget
- No pricing calculator
- No user accounts or login
- No "army of agents" language
- No crosshairs, robots, or military imagery
- No emoji anywhere on the site
