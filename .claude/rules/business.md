## Company architecture
Sonar is a closed-loop intelligence system with two modes:
- LISTEN — always-on capture from web, CRM, social, hiring, product, visitors, account movement
- INTELLIGENCE CORE — resolve identity, infer context, score relevance, select plays, route action, record outcomes
- PULSE — directed activation around a vertical, account set, event, or hypothesis

Revenue is the first packaged system. The architecture supports any high-value workflow.

## ICP (first wedge)
- Founder-led B2B SaaS and tech-enabled services
- 20–150 employees
- $15K+ ACV
- Existing CRM + outbound stack (HubSpot, Apollo, Sales Navigator)
- Inconsistent pipeline despite having the tools

Trigger conditions: new revenue leader hired, SDR/AE/RevOps hiring, funding or expansion, product launch, dormant CRM activity

NOT a fit: no clear ICP, very low ACV, no sales owner, no CRM discipline, wants "more email volume"

## Offer architecture

### Deploy Sonar (revenue teams)
Signal architecture + data sources + context/scoring/"why now" logic + research agents + CRM tasks/alerts/sequences/reporting + managed monitoring and improvement

### Build with Sonar (custom workflows)
Workflow diagnosis + custom agents + integrations + human approvals + ongoing operating support + reusable system components

## Pricing logic
Price the system, not the agent. Clients buy a working workflow, not tokens or agent counts.

- Architecture Sprint: $7,500–$15,000 / 2–4 weeks — map workflow, data, signals, play logic, launch path
- System Deployment: $15,000–$50,000+ / setup — connect tools, agents, CRM, scoring, routing, activation
- Managed Operation: $4,000–$10,000/mo — monitor, maintain, optimize, improve

## What not to build yet
- A complete multi-agent platform before a client buys
- Multiple vertical playbooks at once
- A complex dashboard without a real user
- A large content engine before outbound is running
- Custom features with no reusable value
